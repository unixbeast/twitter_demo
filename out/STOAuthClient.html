Source:
  • st-oauth.js, line 59
Example
st.getAccessToken('some-code', function(error, tokenInfo, smartAppUri) {
   if (!error) {
      // store token and uri somewhere to call later
      // here is what is on the tokenInfo object:
      var accessToken = tokenInfo.access_token;
      var refreshToken = tokenInfo.refresh_token;
      var expiresIn = tokenInfo.expires_in;
      var expiresAt = tokenInfo.expires_at;
   }
});

tokenNeedsRefresh(token, buffer) → {Boolean}

Checks if the token needs to be refreshed. A token is considered as needing to be refresh if it will expire in 60 seconds or less from now, unless otherwise specified.
Parameters:
Name Type Description
token Object The token object as returned by getAccessToken
buffer Number the amount of time in seconds to buffer the refresh token required. Defaults to 60 seconds if not specified.
Source:
  • st-oauth.js, line 329
Returns:
true if the token needs to be refreshed, false if it does not.
Type
Boolean
Example
var expiresWithinOneDay = st.tokenNeedsRefresh(tokenObject, 60 * 60 * 24);

// one minute check is the default:
var expiresWithinOneMinute = st.tokenNeedsRefresh(tokenObject);
